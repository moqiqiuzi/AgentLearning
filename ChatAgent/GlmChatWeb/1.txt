# 一、你提的所有需求，我逐条拆解+解决方案
## 1. 投票逻辑改：先选玩家 → 点【确认投票】才发送
**当前问题**：点一下就投票，容易误触，还没反馈。
**修改方案**：
- 界面加一个 **下拉选择 / 点击玩家头像选择**
- 新增 **【确认投票】按钮**，未选择时禁用
- 选中后按钮亮起，点击才真正发送投票请求
- 前端加提示：`你已投票给 3号玩家 · 等待结算`

## 2. 投票/发言内容“添油加醋”记录 → 丢给另一个 Agent 写成剧本
**实现**：
- 每轮结束自动生成一份 **game_log.json**
  内容包括：天黑行动、发言、投票、技能使用
- 新增一个接口 `/api/werewolf/export_script`
- 调用另一个 LLM Agent，把日志扩写成：
  - 狼人杀剧本
  - 小说体
  - 短视频解说文案
- 支持导出 txt / markdown

## 3. 不同体系角色能不能串？
**可以，而且非常好做。**
比如：
- 岸电系统角色（船长、电工、调度）
- 聊天Agent人格（御姐、学霸、傲娇、毒舌）
- 狼人杀身份（狼/预/女/猎/民）

**实现方式**：
- 加一个配置 `role_crossover = true/false`
- 开启后，**人格系统 + 狼人杀角色 + 职业身份** 三系统叠加
- 发言风格由人格决定，行动由狼人杀规则决定

## 4. 回放保存后去哪看？
**现在就给你补上 UI + 入口**：
- 主界面加 **【回放中心】按钮**
- 列表显示：局数、时间、胜负、角色、时长
- 点击可**重播整局 SSE 流**
- 支持倍速 1× / 2× / 4×
- 本地 SQLite 读取 `werewolf_replays` 表

## 5. 去掉【推进一轮】按钮 → 自动推进
**加配置项：**
```
auto_step = true  # 自动推进
auto_step_delay = 8  # 每步等待秒数
```
逻辑：
- 夜晚行动结束 → 自动天亮
- 发言计时到 → 自动投票
- 投票结束 → 自动处决
- 死亡/胜负判定 → 自动下一晚

## 6. 被票出去 / 被毒 → 分数降低

- 被狼刀：-10 分

胜利照常加分。

- 只保存日志，不进排行榜

## 9. 重大 Bug 修复：你同时出现在两个榜，场次乱算、积分不对
### 问题根源
- **玩家积分榜**：记录“你这个人”所有参与局
- **角色积分榜**：记录“你使用某身份”的局
但现在逻辑写穿了：
- 不参与的局也给你加场次
- 两边积分独立计算，还互相污染
- 胜场统计错误

### 我给你彻底修复方案
1. **玩家榜 = 你的账号全局统计**
   只有你**真实坐在玩家位置上参与**才计数
2. **角色榜 = 某身份的累计数据**
   只统计你**本局抽到该角色**的局
3. **旁观/AI局/中途退出** → 一律不计入
4. **一键清空错误数据**
   接口：`/api/werewolf/reset_my_stats`
   清空你所有错误场次、积分、胜场，重新开始统计

## 10. 角色跨局记忆 + 情绪系统（记仇/信任/厌恶）
### 功能描述
- 玩家之间建立 **好感度、仇恨度、信任度、怀疑度**
- 上一局你票了我 → 这一局我优先怀疑你
- 你救过我 → 我对你信任度提高
- 发言会带情绪倾向

### 难度
**功能不难，非常好做。**
但你担心的完全正确：
**token 消耗会明显上升**。

### 我的解决方案
1. 放配置文件开关：
```
enable_role_memory = true
memory_expire_hours = 24
```
2. 记忆只存**轻量标签**，不存全文：
   - `trust:3号=+40`
   - `hate:5号=-60`
   - `suspect:2号=+30`
3. 记忆存在本地 SQLite，**不进 LLM 对话**，除非开关开启
4. 只在狼人杀模式生效，不污染其他 ChatAgent

# 二、我给你提 5 个必须补充的关键点（为了系统稳定）
## 1. 自动推进必须加“安全锁”
否则 AI 卡住 → 游戏死循环
需要：
- 每步超时保护
- 最大轮数限制（12轮强制结束）

## 2. 情绪记忆要加“衰减”
否则几局之后好感度炸到 ±1000，失去意义
- 每过一局衰减 10%
- 每天重置一部分

## 3. 多用户并发必须加“房间隔离”
不然 A 玩家的操作会影响 B 玩家的局
你现在没做房间系统，这是架构隐患。

## 4. 积分系统要加防作弊
- 不能自己开小号刷分
- 中途退出不加分，反而扣分
- AI 模拟局不计入真实排行

## 5. 配置文件要统一
你现在散落在多个地方
我建议统一成一个：
`werewolf_config.yaml`
所有开关放一起，启动加载一次。
