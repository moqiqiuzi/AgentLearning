📖 狼人杀游戏规则

📋 基本规则

游戏分为好人阵营和狼人阵营。夜晚狼人杀人，白天所有人讨论并投票放逐一人。

🏆 胜利条件

• 好人胜：所有狼人被淘汰
• 狼人胜：存活狼人 ≥ 存活好人

🎭 角色说明

🐺 狼人：每晚与队友商议杀一人
🔮 预言家：每晚查验一人的身份（好人/狼人）
🧪 女巫：有一瓶解药（救人）和一瓶毒药（毒人），各用一次
🏹 猎人：被杀或被投票出局时可带走一人
🛡️ 守卫：每晚守护一人，不能连续两晚守护同一人
👤 村民：无特殊能力，靠推理投票

🔄 游戏流程

1. 夜晚：守卫守护 → 狼人杀人 → 预言家查验 → 女巫用药
2. 天亮：公布死亡 → 遗言 → 发言讨论 → 投票放逐
3. 循环直到一方获胜

🎮 操作说明（玩家模式）

• 夜晚：点击侧边栏玩家卡片选择目标，然后点确认
• 白天发言：底部输入框打字或点「AI帮我发言」
• 投票：点击侧边栏目标玩家，点「确认选择」

📊 积分规则

• 胜利基础分：狼人+150 / 预言家+120 / 女巫+110 / 猎人+110 / 守卫+110 / 村民+80
• 失败：-30分
• 死亡扣分：被投票-40 / 被毒-20 / 被狼刀-10 / 被猎人带-20
• 加分项：猎人带走狼+30 / 预言家正确查验+20/次
• MVP：+50分
